# MATA60-Marco-2

Projeto da disciplina MATA60 – Banco de Dados, contendo os scripts SQL do Marco 1 e Marco 2 para um sistema de controle de eventos acadêmicos. O repositório inclui a criação do esquema, população de dados e rotinas avançadas (materialized views, stored procedures e transações), desenvolvidas exclusivamente em SQL para PostgreSQL, conforme o barema da disciplina.
